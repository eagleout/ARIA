# 🦒 Les Petitous — Page "Nos Crèches"

## SEO
- **URL** : `/nos-creches/`
- **Title** : `Nos micro-crèches en Île-de-France | Les Petitous`
- **Meta description** : `Découvrez les micro-crèches Les Petitous en Île-de-France : Villemomble (93), Juvisy-sur-Orge (91), Kremlin-Bicêtre (94). 14 enfants max, aide CAF. Réservez une visite.`

## 5 blocs à coller dans Elementor

| # | Fichier | Section |
|---|---------|---------|
| 01 | `01-HERO.html` | Hero avec H1, breadcrumb, stats réseau |
| 02 | `02-FICHES-CRECHES.html` | 3 grandes fiches détaillées (1 par crèche) |
| 03 | `03-CARTE.html` | Carte interactive Île-de-France + légende |
| 04 | `04-ENGAGEMENTS.html` | 6 engagements communs à toutes les crèches |
| 05 | `05-CTA.html` | Bandeau CTA final |

## Installation
1. Crée la page `/nos-creches/` dans WordPress
2. Template Elementor : Pleine largeur
3. 5 sections, chacune avec un widget HTML
4. Configure le Title SEO et la Meta description dans Rank Math

## Points importants
- Les fiches crèches (bloc 02) alternent : photo à gauche / photo à droite
- Chaque fiche renvoie vers la page établissement individuelle
- La carte (bloc 03) nécessite un iframe Google My Maps personnalisé
- Le bloc engagements (04) renforce le maillage interne vers /notre-pedagogie/ et /tarifs-aides-financieres/
