# 🦒 Les Petitous — Page "Notre Pédagogie"

## SEO
- **URL** : `/notre-pedagogie/`
- **Title** : `Notre approche pédagogique | Micro-crèche Les Petitous`
- **Meta description** : `Découvrez la pédagogie des crèches Les Petitous : éveil sensoriel, activités Montessori, accompagnement personnalisé. Un environnement pensé pour chaque enfant.`

## 6 blocs à coller dans Elementor

| # | Fichier | Section |
|---|---------|---------|
| 01 | `01-HERO.html` | Hero avec H1, philosophie, citation Montessori |
| 02 | `02-PILIERS.html` | Les 5 piliers pédagogiques (numérotés) |
| 03 | `03-JOURNEE-TYPE.html` | Timeline visuelle d'une journée (9 étapes) |
| 04 | `04-ACTIVITES.html` | Grille de 6 activités avec tags |
| 05 | `05-ALIMENTATION.html` | Section repas faits maison + 4 engagements |
| 06 | `06-CTA.html` | Bandeau CTA "Réserver une visite" |

## Installation
Même procédure :
1. Crée la page `/notre-pedagogie/` dans WordPress
2. Template Elementor : Pleine largeur
3. 6 sections, widget HTML dans chacune
4. Configure Title SEO et Meta description dans Rank Math
